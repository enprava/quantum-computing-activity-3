\chapter{Objetivos}

El objetivo de este Trabajo es identificar un ámbito de aplicación relevante para la computación cuántica y diseñar una propuesta de solución basada en un algoritmo cuántico. Para ello, se plantea un enfoque que permita analizar las posibilidades de mejora que ofrece la computación cuántica frente a métodos clásicos en tareas de procesamiento de información complejas.

\section{Objetivo general}

Diseñar una solución conceptual basada en un algoritmo cuántico, justificando su adecuación al contexto de aplicación elegido y analizando las ventajas que podría aportar respecto a una aproximación clásica.

\section{Objetivos específicos}

\begin{itemize}
	\item Identificar un contexto de aplicación en el que la computación cuántica pueda aportar una ventaja significativa.
	\item Formular los requisitos generales que debe cumplir la solución propuesta.
	\item Seleccionar un algoritmo cuántico adecuado para el tipo de tarea planteada.
	\item Describir de forma estructurada el funcionamiento de la solución basada en dicho algoritmo.
	\item Comparar conceptualmente la propuesta cuántica con un enfoque clásico equivalente.
	\item Evaluar las limitaciones, ventajas y condiciones necesarias para su posible aplicación práctica.
\end{itemize}
